Mixins

apiResourceList

APIResourceList is a list of APIResource, it is used to expose the name of the resources supported in a specific group and version, and if the resource is namespaced.

    groupVersion is the group and version this APIResourceList is for.

  • withResources 

    withResourcesfunction(resources) self + if std.type(resources) == 'array' then { resources: resources } else { resources: [resources] }

    resources contains the name of the resources and if they are namespaced.

    withResources takes an array of type resourcesType. You can create an instance of resourcesType with hidden.meta.v1.apiResource.new().
