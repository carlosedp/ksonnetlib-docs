statefulSetStatus

StatefulSetStatus represents the current state of a StatefulSet.

Functions
  • withCollisionCount 

    withCollisionCountfunction(collisionCount) self + { collisionCount: collisionCount }

    collisionCount is the count of hash collisions for the StatefulSet. The StatefulSet controller uses this field as a collision avoidance mechanism when it needs to create the name for the newest ControllerRevision.

  • withConditions 

    withConditionsfunction(conditions) self + if std.type(conditions) == 'array' then { conditions: conditions } else { conditions: [conditions] }

    Represents the latest available observations of a statefulset’s current state.

    withConditions takes an array of type conditionsType. You can create an instance of conditionsType with hidden.apps.v1beta2.statefulSetCondition.new().

    see hidden.apps.v1beta2.statefulSetCondition

  • withCurrentReplicas 

    withCurrentReplicasfunction(currentReplicas) self + { currentReplicas: currentReplicas }

  • withCurrentRevision 

    withCurrentRevisionfunction(currentRevision) self + { currentRevision: currentRevision }

    currentRevision, if not empty, indicates the version of the StatefulSet used to generate Pods in the sequence [0,currentReplicas).

  • withObservedGeneration 

    withObservedGenerationfunction(observedGeneration) self + { observedGeneration: observedGeneration }

    observedGeneration is the most recent generation observed for this StatefulSet. It corresponds to the StatefulSet’s generation, which is updated on mutation by the API Server.

  • withReadyReplicas 

    withReadyReplicasfunction(readyReplicas) self + { readyReplicas: readyReplicas }

    readyReplicas is the number of Pods created by the StatefulSet controller that have a Ready Condition.

  • withReplicas 

    withReplicasfunction(replicas) self + { replicas: replicas }

  • withUpdateRevision
