  • resourceAttributes

    ResourceAuthorizationAttributes describes information for a resource access request

    Functions
    • mixinInstance 

      mixinInstancefunction(resourceAttributes) __resourceAttributesMixin(resourceAttributes)

    • withGroup 

      withGroupfunction(group) self + __resourceAttributesMixin({ group: group })

      Group is the API Group of the Resource. “*” means all.

    • withName 

      withNamefunction(name) self + __resourceAttributesMixin({ name: name })

      Name is the name of the resource being requested for a “get” or deleted for a “delete”. “” (empty) means all.

    • withNamespace 

      withNamespacefunction(namespace) self + __resourceAttributesMixin({ namespace: namespace })

      Namespace is the namespace of the action being requested. Currently, there is no distinction between no namespace and all namespaces “” (empty) is defaulted for LocalSubjectAccessReviews “” (empty) is empty for cluster-scoped resources “” (empty) means “all” for namespace scoped resources from a SubjectAccessReview or SelfSubjectAccessReview

    • withResource 

      withResourcefunction(resource) self + __resourceAttributesMixin({ resource: resource })

      Resource is one of the existing resource types. “*” means all.

    • withSubresource 

      withSubresourcefunction(subresource) self + __resourceAttributesMixin({ subresource: subresource })

      Subresource is one of the existing resource types. “” means none.

    • withVerb 

      withVerbfunction(verb) self + __resourceAttributesMixin({ verb: verb })

      Verb is a kubernetes resource API verb, like: get, list, watch, create, update, delete, proxy. “*” means all.

    • withVersion 

      withVersionfunction(version) self + __resourceAttributesMixin({ version: version })

      Version is the API Version of the Resource. “*” means all.
